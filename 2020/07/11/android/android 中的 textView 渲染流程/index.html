知识点：为什么不能在主线程更新 UI

设计考虑：如果在更新 UI 时需要考虑多线程安全问题的话，会导致性能降低，为了避免这种性能降低，所以规定只能在 UI 线程中更新界面，如果不在主线程中做更新界面的实现的话，就会主动抛出异常。
